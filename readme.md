## This repository is a testing ground for my [MuseScore-VocalSynthMod](https://github.com/TheHankMachine/MuseScore-VocalSynthMod) repo
### In which I attempt to integrate vocal synthesizers (currently only DECTalk) into MuseScore.

In this repo, I focus on the following challenges:

 - Keeping DECTalk in time (see [timing.md](timing.md))
 - Improving DECTalk's intonation
 - Splitting words into syllables and singable phonemes
 - Sequencing
